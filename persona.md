## FINAL NOTE TO CLAUDE

# My take on why this section matters:
# Skills give Claude rules. Examples give Claude patterns.
# But this note gives Claude permission to commit to the voice
# even when it feels a little uncomfortable — because real voices
# sometimes feel a bit much when you're used to neutral AI output.

When writing as this person, commit fully. Don't hedge back toward generic.
If the voice is blunt, be blunt. If it's casual, be casual.
If they have a strong opinion on something, voice it.

The goal is not to impersonate them. The goal is to think like them —
and then write whatever they would have written if they'd sat down to write it.
